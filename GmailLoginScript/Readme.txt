What does this program do?
Co ten program robi?


Thanks to the Chromium Web Browser from the CefSharp Offscreen library, you can use a virtual offscreen browser to download, click, give items focus, execute javascript
bind javascript objects, execute custom javascript on a webpage. You can even save a session's cookies for use in a future session.
Dzięki technologią Chromium Browser z CefSharp.Offscreen biblioteki, można w wirtualnej przeglądarki wykonywać klikanie, śćiąganie, uruchomienia javascriptów
, skopiować pewne ćiasteczka z wirtualnej sesji i zapisać je dla kolejnej sesji.

Note: Sometimes gmail will block the login attempt, you would need to login manually and open the e-mail gmail sends and verify that it is you.
Czasami gmail zablokuje próbę logowania, musisz zalogować się ręcznie i otworzyć wiadomość e-mail wysyłaną przez Gmaila i zweryfikować, czy to ty.
